Análisis de Potenciar Solidario

En el marco del curso Data Analytics ofrecido por Guayerd & IBM, realicé una análisis descriptivo para la ONG Potenciar Solidario. Se utilizaron herramientas de Google, como Sheets y Looker Studio, para el procesamiento de datos y la creación de un Dashboard.
El presente trabajo contiene indicadores claves para el tipo de organización, como cantidad de donaciones recibidas, cantidad de donantes registrados, tota de ingresos y egresos, entre otras. El objetivo era crear una herramienta de apoyo para la toma de decisiones informadas.

Con respecto a la usabilidad del dashboard, se destacan :

    - Los filtros de Año, Mes o Trimestre para que el usuario pueda fijar la información por períodos de tiempo

    - Botón de cambio de tema predeterminado. Funcionalidad que pone enfoque en la experiencia de usuario, ya que éste puede elegir entre tema claro u oscuro.

    - Botón para restablecer los filtros. De esta manera el usuario no debe hacerlo manualmente en cada uno.

    - Botones de navegación. Permite al usuario la navegación entre páginas.

    - Gráficos de varias capas. Agrega interactividad a las visualizaciones. El usuario puede pasar de una gráfica anual a una mensual sin necesidad de un segmentador particular.
